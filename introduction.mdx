## AI Intelligent Document Processing (IDP)

Our AI based intelligence document processing, IDP, engine collects information from documents
and turns them into valuable insights.

It's used by us to create valuable datasets but you can also use the same set of API to extract meaningful
information from unstructured data for your own documents.

Read more in the [introduction](/api-idp/introduction).
