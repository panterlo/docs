Our API follows OpenAPI specification 3.0.1. This means you can use preferrable use the [OpenAPI generator](https://openapi-generator.tech/)
to generate your client.

## Definition

The OpenAPI definition endpoint is https://api.tic.io/docs/v1/swagger.json
