[Finansinspektionen](https://www.fi.se/en/) is Sweden's government authority tasked with monitoring the financial market.

This dataset has been tracking data from Finansinspektionen since 2016.

## What's included

This dataset covers daily and second updates of:

- Notifications / Flagging regulatory events for listed companies
- Insider trading with easy to use summary of the instruments
- Public registry information about listed companies
- Fund holdings quarterly updates

## How to gain access

To access this dataset the user must have claims including finansinspektionen.
